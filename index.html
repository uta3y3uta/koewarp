🔍 記録先の接続テスト

受け取り口の作り方（初回だけ・約3分）
  1. 保存先フォルダを用意する
    Googleドライブで音声をためるフォルダを作成。 そのフォルダの共有URLをコピーします。
  2. フォルダURLをここに貼り付ける
    未入力
  3. コードをコピーする
    フォルダIDが自動で埋め込まれます
  4. Apps Scriptに貼り付ける
    1. Apps Scriptで新規プロジェクトを開く
    2. 画面の既存コード（function myFunction…）を全部消す（Ctrl+A → Delete）
    3. Ctrl+Vで３のGASコードを貼り付け
    4. 💾保存
  5. デプロイして公開する
    1. 右上の青い「デプロイ」→「新しいデプロイ」
    2. 左の⚙（歯車）→「ウェブアプリ」を選択
    3. 「次のユーザーとして実行」＝自分
    4. 「アクセスできるユーザー」＝全員 → 「デプロイ」
    5. 初回は承認画面が出ます。自分のアカウントを選択 →「詳細」→「（安全ではないページ）に移動」→「許可」
  6. URLを貼り付けて完成
    表示された「ウェブアプリ」のURL（末尾が…/exec）をコピーし， このページ上部の記録先欄に貼り付け，共有URLを発行したら完了！
